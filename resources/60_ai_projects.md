## How to Use This Guide

- Pick a category that matches your current interest or job target (RAG, agents, multimodal, etc.).
- For each project:
  - Read the **Description** and **Skills Gained**.
  - Follow the **Tutorial** and **Code** links.
  - Build your own implementation and link it under **My Implementation**.
- Update the **Status** so you can track your progress over time.

I add my own notes, improvements, and extra links as I go.

## Project Format

Each project follows this structure:

- **Title**
- **Difficulty Level** (1–5)
- **Description**
- **Skills Gained**
- **Resources** (Tutorial, Code, Models, etc.)
- **My Implementation** (your GitHub link)
- **Status**: ⬜ Not started / 🟡 In progress / ✅ Done

Below I keep the original project ideas but adapt them into this consistent format and add my own notes over time.
